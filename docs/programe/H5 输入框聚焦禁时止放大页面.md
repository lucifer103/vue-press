# HTML5 输入框聚焦时禁止放大页面

1. 找到项目的入口文件：`index.html`；

2. 在页面的 `<meta name="viewport"/>` 的标签最后添加 `user-scalable=no` 即可，如：

    `<meta name="viewport" content="width=device-width,initial-scale=1.0,user-scalable=no">`
