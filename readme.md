# VuePress Tool

## VuePress 博客搭建工具

基于 [Yur](https://github.com/cnguu/vuepress-theme-yur) 主题的 VuePress 博客快速搭建工具

## 准备

- :octocat:源码仓库（源码备份）

## 开发环境

- Git
- Node.js
- yarn

## 使用

1. 下载：`git clone https://github.com/ioim/vuepress-theme-yur-tool.git`
2. 删除 `.git` 目录
3. 安装：`npm install`
4. 本地预览：`npm run start`
5. 编译：`npm run build`

### 部署

需要修改`deploy.sh`脚本
